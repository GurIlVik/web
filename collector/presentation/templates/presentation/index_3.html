Парадная

 

 

Для не зарегистрированного посетеителя реализовано 2а сценария:

1. вход на главную страницу без регистрации

2. регистрация

 

Для члена клуба не аутентифицированного 3и сценария

1. вход на главную страницу без регистрации

2. аутентифицикация - по паролю

3. аутентифицикация - по паролю присланному по почте (в случае если член клуба забыл пароль)

 

Для аутентифицированного 3и сценария

1. выход

2. переход в зал для общения

3. переход в личный кабинет

 

 

Регистрация 2х этапная:

на 1ом - вводятся: псевдоним, пароль 2а раза, и личный почтовый ящик

на 2ом - подтверждение почты с присланного на нее сгенерированного случайным образом токена

 

Аутентификация по почте 2х этапная:

на 1ом - вводятся: почтовый ящик

на 2ом - вводится сгенерированный случайным образом токен - впоследствии служащий паролем

 

Аутентификая с паролем - стандартная
